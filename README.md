# PGADMIN4 
## Инструкция
1. Клонируем репозиторий в папку www-data:
```sh
git clone https://github.com/nindeco/pgadmin4 
```
2. Заходим в папку:
```sh
cd pgadmin4
```
3. Выполняем команду:
```sh
docker-compose up -d
```
___
### Доступы
- Доступ к pgadmin по ссылке: http://localhost
- Пользователь: admin
- Пароль: defender
___
### Про дампы
- Все сделанные дампы лежат по пути: pgadmin4/pgadmin/data
___
